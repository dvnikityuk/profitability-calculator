# 📊 Калькулятор рентабельности продаж РМ

Интерактивный калькулятор для расчета маржинальной доходности (МД) и маржинальной рентабельности (МР%) продаж.

![Status](https://img.shields.io/badge/status-ready-success)
![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ Возможности

- 🇧🇾🇷🇺 **Два рынка**: Беларусь и Россия с раздельными данными
- 📁 **Загрузка CSV**: Импорт прайс-листов и TVC из отдельных файлов
- 🔍 **Фильтрация**: Поиск по классам ТМЦ, названиям и кодам
- 📈 **Расчеты**: Автоматический расчет МД, МР%, отклонений
- 💰 **Гибкое ценообразование**: Скидки и наценки с визуальной индикацией
- 📤 **Экспорт**: Выгрузка результатов в CSV
- 🎨 **Современный UI**: Адаптивный дизайн с цветовой индикацией рентабельности

## 📁 Форматы файлов

### Прайс-лист (CSV)
```csv
Прайс,Класс ТМЦ,ТМЦ кратко,Цена,ТМЦ,Код
РОССИЯ,Материал для маркировки,Лента этикетировочная,1354.00,Лента этикетировочная «КОМПО»,053360
БЕЛАРУСЬ,Материал для маркировки,Лента этикетировочная,1200.00,Лента этикетировочная «КОМПО»,053360
```

**Колонки:**
- `Прайс` — рынок (БЕЛАРУСЬ или РОССИЯ)
- `Класс ТМЦ` — категория товара
- `ТМЦ кратко` — краткое название
- `Цена` — цена в рублях (разделитель дробной части: точка или запятая)
- `ТМЦ` — полное название
- `Код` — уникальный код товара (ключ для соединения с TVC)

### TVC (CSV)
```csv
Класс ТМЦ,ТМЦ кратко,TVC,ТМЦ,Код
Материал для маркировки,Лента этикетировочная,8.21,Лента этикетировочная «КОМПО»,053360
```

**Колонки:**
- `Класс ТМЦ` — категория товара
- `ТМЦ кратко` — краткое название
- `TVC` — переменные затраты на единицу в евро
- `ТМЦ` — полное название
- `Код` — уникальный код товара (должен совпадать с кодом в прайсе)

## 🎯 Как использовать

1. **Загрузите данные**
   - Нажмите кнопку «Загрузить данные»
   - Загрузите файл с TVC (общий для всех рынков)
   - Загрузите файл с прайсом (с указанием рынка)

2. **Настройте параметры**
   - Установите курс €/₽
   - При необходимости укажите скидку дилера

3. **Отберите позиции**
   - Используйте фильтры для поиска нужных товаров
   - Добавьте позиции в таблицу

4. **Рассчитайте рентабельность**
   - Введите количество
   - Укажите скидку или наценку (или конечную цену)
   - Система автоматически рассчитает МД и МР%

5. **Экспортируйте результаты**
   - Нажмите «Выгрузить CSV» для сохранения расчетов
