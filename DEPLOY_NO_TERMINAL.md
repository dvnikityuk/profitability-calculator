# 🚀 Публикация на GitHub Pages — без терминала

Эта инструкция поможет вам опубликовать калькулятор рентабельности на GitHub Pages **без использования терминала** после первоначальной настройки.

---

## 📋 Шаг 1: Создайте репозиторий на GitHub

1. Откройте [https://github.com/new](https://github.com/new)
2. Введите имя репозитория, например: `profitability-calculator`
3. Выберите **Public** (бесплатно для GitHub Pages)
4. **НЕ ставьте** галочку "Initialize with README"
5. Нажмите **Create repository**

---

## 📋 Шаг 2: Установите GitHub Desktop

1. Скачайте с [https://desktop.github.com/](https://desktop.github.com/)
2. Установите и войдите в свой GitHub аккаунт

---

## 📋 Шаг 3: Добавьте проект в GitHub Desktop

1. Откройте GitHub Desktop
2. Нажмите **File → Add Local Repository**
3. Выберите папку с вашим проектом
4. Если появится сообщение "This directory does not appear to be a Git repository" — нажмите **create a repository**
5. В поле Name введите имя (например, `profitability-calculator`)
6. Нажмите **Create Repository**

---

## 📋 Шаг 4: Первый коммит

1. В GitHub Desktop внизу слева вы увидите все файлы проекта
2. В поле **Summary** (внизу слева) напишите: `Initial commit`
3. Нажмите синюю кнопку **Commit to main**

---

## 📋 Шаг 5: Опубликуйте репозиторий

1. В GitHub Desktop нажмите **Publish repository** (синяя кнопка сверху)
2. Убедитесь что:
   - Имя правильное
   - **Keep this code private** — **НЕ** ставьте галочку (GitHub Pages работает только для public репозиториев на бесплатном тарифе)
3. Нажмите **Publish repository**

---

## 📋 Шаг 6: Включите GitHub Pages

1. Откройте ваш репозиторий на GitHub.com
2. Перейдите во вкладку **Settings** (Настройки)
3. В левом меню найдите раздел **Pages**
4. В разделе **Build and deployment**:
   - **Source**: выберите `GitHub Actions`
5. Нажмите **Save**

---

## 📋 Шаг 7: Автоматический деплой

Теперь при **каждом обновлении** файлов проекта:

1. Откройте GitHub Desktop
2. Введите описание изменений в поле **Summary**
3. Нажмите **Commit to main**
4. Нажмите **Push origin** (синяя кнопка сверху)

**Готово!** Через 1-2 минуты ваш сайт обновится автоматически.

---

## 📋 Шаг 8: Проверьте результат

Ваш калькулятор будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/ВАШ_РЕПОЗИТОРИЙ/
```

Например:
```
https://ivanov.github.io/profitability-calculator/
```

---

## 🔧 Если что-то пошло не так

### Проблема: Сайт не открывается
- Подождите 2-3 минуты после пуша
- Проверьте вкладку **Actions** в репозитории — там должен быть зелёный кружок ✅
- Если красный ❌ — нажмите на него, посмотрите ошибку

### Проблема: Actions не запускается
- Убедитесь что файл `.github/workflows/deploy.yml` существует в репозитории
- Попробуйте сделать ещё один коммит и пуш

### Проблема: Старая версия сайта
- GitHub Pages кэширует — подождите 5 минут или нажмите Ctrl+F5 в браузере
